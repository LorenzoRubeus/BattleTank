# BattleTank
An open world head-to-head tank fight with Ai, terrain, control system in UE4

## Requirements
### SFX: 
* Gun Firing
* Explosion
* Barrel Moving
* Turret Moving
* Engine Sound
### Static Mesh:
* Simple tank comprising tracks
* Body
* Turrets
* Barrels
### Textures:
* Adding for visual flare
### Music:
* Background music to create tension
